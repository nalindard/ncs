# No Copywrite Sounds alternate frontend

---

### Features

1. Search for songs by:
    1. name
    2. genre
    3. mood
2. Download songs.
3. Play songs.
4. Add songs to current list.
5. Seek tracks.
6. Repeat.
7. Suffel.
8. Volume control.
